# stand-with-israel

Support the people of Israel as they face the aftermath of the atrocities committed by Hamas on October 7th

```html

      src="https://covverapp.github.io/stand-with-israel/"
      style="
        width: 100%;
        height: 400px;
        border: 0;
        overflow: hidden;
      "
      title="we-stand-with-israel"
></iframe>
```
